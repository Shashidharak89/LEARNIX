Learnix

A lightweight learning-content sharing platform — Next.js + MongoDB

Quick Links

Open these routes locally while the dev server runs:

  • /dashboard — Dashboard
  • /login — Login / Register
  • /search — Search
  • /learn — Learn
  • /works — Uploaded Works
  • /upload — Upload (includes Updates banner)
  • /upload/updates — Manage Updates
  • /materials — Study Materials
  • /qp — Question Papers
  • /tools — Tools (TextShare, converters)
  • /feedback — Feedback
  • /help — Help

Features

  • Upload & manage subjects, topics and student works
  • Study materials browsing and question papers
  • TextShare editor with local storage 'My Codes'
  • Chunked file upload (frontend)
  • Updates activity feed (create/edit/delete announcements)

Tech Stack

Core technologies used

Next.js React MongoDB Mongoose Cloudinary CSS

Quick Start

npm install
npm run dev
# open http://localhost:3000

Developer Notes

  • Auth currently uses `localStorage.usn` to resolve server user ids. Consider JWT or sessions.
  • Chunked upload backend endpoints are pending (frontend exists).
  • Styles live under src/app/.../styles. AddUpdateForm has a CSS file under the upload updates styles.
